Simple start command line(Ubuntu)

For connecting you can use address without kaspa: prefix, some mining OS can't configure flight sheet correctly because of this prefix
./kaspa-miner -s stratum+tcp://acc-pool.pw:16061 -a WALLET_ADDRESS
./bzminer -a kaspa -w WALLET_ADDRESS -p stratum+tcp://acc-pool.pw:16061
./SRBMiner-MULTI --disable-cpu --algorithm kaspa --pool ru.acc-pool.pw:16061!hk.acc-pool.pw:16061!ca.acc-pool.pw:16061!acc-pool.pw:16061 --wallet WALLET_ADDRESS --password x
./lolMiner --algo ETHASH --pool eu1.ethermine.org:4444 --user WALLET_ADDRESS --worker lolMinerWorker --dualmode KASPADUAL --dualpool acc-pool.pw:16061 --dualuser WALLET_ADDRESS --dualworker lolMinerWorker --apiport 8020 --watchdog exit
